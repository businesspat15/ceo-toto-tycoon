# CEO-toto-tycoon Deployment Guide

This document explains how to deploy the **CEO-toto-tycoon** project, including both the **frontend** (React + Vite) and **backend** (Node.js + Express).

---

## Prerequisites

Make sure you have installed:

- [Node.js](https://nodejs.org/) v18+  
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)  
- A PostgreSQL database (or Supabase) for user data  
- Telegram WebApp credentials (if using Telegram integration)
